Class: RCObject

RCObject(name, gridX, gridY, width, height, objectClass, userClass, tocDepth)

Abstract class specfiying basic functionality of all RCObjects, which are subclasses of RCObject.

Constructor

new RCObject(name, gridX, gridY, width, height, objectClass, userClass, tocDepth)

Creates an RCObject. Cannot be called directly, but only by constructors of subclasses.
Parameters:
Name Type Default Description
name string the name of the tool
gridX number horizontal position
gridY number vertical position
width number 1 number of cells it spans horizontally
height number 1 number of cells it spans vertically
objectClass string CSS class for object
userClass string CSS class specified by user
