Class: RCMedia

RCMedia(name, gridX, gridY, url, pxWidth, pxHeight, width, height)

Abstract class for media subclasses for image, svg, pdf, video, audio

Constructor

new RCMedia(name, gridX, gridY, url, pxWidth, pxHeight, width, height)

Creates an RCMedia object. Cannot be called directly, only by the constructors of subclasses.
Parameters:
Name Type Default Description
name string the name of the tool
gridX number horizontal position
gridY number vertical position
url string the file to be displayed
pxWidth number the width in pixels
pxHeight number the height in pixels
width number 1 number of cells it spans horizontally
height number 1 number of cells it spans vertically
